# Portfolio

Personal portfolio website built with **React** and **Framer Motion**, featuring a dark, editorial design and smooth scroll-based animations.

## Overview
- Single-page layout
- Dark, minimal UI
- Smooth, restrained animations
- Responsive and performance-focused

## Tech Stack
- React
- Vite
- Framer Motion
- CSS (custom properties)

## Getting Started

Install dependencies:
```bash
npm install
```
